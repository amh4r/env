# Sessions

`<prefix>C` -- New session (with name prompt)
`<prefix><C-f>` -- Switch session (fzf)

# Windows

`<prefix>c` -- New window (with name prompt)
`<prefix>,` -- Rename window
`<prefix><C-h>` -- Previous window
`<prefix><C-l>` -- Next window
`<M-Tab>` -- Toggle last window
`<prefix><S-Left>` -- Reorder window left
`<prefix><S-Right>` -- Reorder window right

# Panes

`<prefix>\` -- Vertical split
`<prefix>-` -- Horizontal split
`<prefix>h` -- Move left
`<prefix>j` -- Move down
`<prefix>k` -- Move up
`<prefix>l` -- Move right
`<prefix>H` -- Resize left
`<prefix>J` -- Resize down
`<prefix>K` -- Resize up
`<prefix>L` -- Resize right

# Other

`<prefix>r` -- Reload config
